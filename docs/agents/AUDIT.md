## Формат записи

```
ДАТА: YYYY-MM-DD
TC: TC-XXX (или ручное)
АВТОР: 007 / Центр / Никита
ИЗМЕНЕНО:
- файл/зона — что конкретно
РИСК: low / medium / high
ПОЧЕМУ: архитектурное обоснование
ПРОВЕРЕНО:
- [ ] tsc --noEmit
- [ ] vitest run --related
- [ ] manual test
- [ ] 009 runtime check
ДОКУМЕНТАЦИЯ:
- какие документы обновлены
ОТКАТ: как откатить если что-то сломалось
```
