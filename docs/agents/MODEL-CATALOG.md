| **T0: FREE** | GLM-5.1 (opencode) | Разведка, рутина, все скауты | $0 (включено в opencode) | Всегда |
| **T1: MICRO** | DeepSeek V3 | Глубокий анализ, центр | $0.14 in / $0.28 out | Еженедельно |

| **T4: HEAVY** | Claude 3.5 Sonnet | Критические архитектурные решения | $3 in / $15 out | 1-2 раза/мес |

## Маршрутизация через OpenRouter Gateway

Существующий `gateway/src/index.ts` уже проксирует к OpenRouter. Схема:

```
Субагент → OpenCode → 007 маршрутизатор → gateway/v1/chat/stream → OpenRouter → Модель
                                                                          ↓
                                                                     GLM-5.1 (бесплатно, локально)
                                                                     DeepSeek V3 (дешёво)
                                                                     GPT-4o-mini (легко)
                                                                     Kimi K2.6 (контекст)
                                                                     Claude 3.5 (тяжело, редко)
```

### Как переключать модель

В теле запроса к gateway указываем `model`:
```json
{
  "model": "deepseek/deepseek-chat-v3-0324",
  "messages": [...],
  "stream": true
}
```

Доступные модели через OpenRouter:
- `deepseek/deepseek-chat-v3-0324` — DeepSeek V3
- `openai/gpt-4o-mini` — GPT-4o-mini
- `anthropic/claude-3.5-sonnet` — Claude 3.5 Sonnet
- `moonshotai/kimi-k2` — Kimi K2 (если доступен)

## Оптимизация стоимости

1. **Батчинг:** Скауты запускаются группами, не индивидуально
2. **Кэширование:** Одинаковые запросы → Redis/кэш (уже есть в gateway)
3. **Приоритет vs цена:** Routine = GLM-5.1, Deep = DeepSeek, Critical = Claude
4. **Контекст-окна:** Kimi K2.6 для документов > 50K символов
5. **Токен-бюджет:** Каждый скаут → лимит 10K токенов/сессию
