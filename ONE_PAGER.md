# beLive — One‑Pager

Mission
Create an open, web-first platform for musicians to rehearse, perform and create content: realtime rehearsal loop + vocal tools + visual concert mode. Open-source with premium services.

Problem
Musicians need lightweight, low‑friction tools to rehearse, isolate stems, practice with loops and run small visual concerts — most solutions are heavy, desktop-only or costly.

Solution
beLive — web app with real‑time rehearsal loops, vocal/instrument isolation, tempo/pitch controls with high quality, and a visual concert mode. Web-first (no install), modular audio engine, and AI assistant for vocal coaching.

Key features (MVP)
- Realtime rehearsal loop + tempo controls
- Vocal/instrument isolation and simple stem export
- Visual concert/player mode
- Mobile-first UI and low-latency web audio
- Freemium model: core free, paid stems/HQ export, private rooms, AI features

Traction / assets
- Public demo (GitHub Pages) with 2–3 short demo videos
- ROADMAP + open-source repo on GitHub
- Early interest list / pilot offer for studios/artists

Business model
- Freemium + paid exports / HQ downloads
- Creator monetization (tips/tickets) with platform fee
- B2B: white‑label / SDK for studios and DAWs

Team & ask
- Small founding team (dev + product + creative)
- Current ask: introductions to 5–10 music-industry angels / studio leads, or pre-seed $100k to $300k to build paid features and pilots.
