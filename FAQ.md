# Frequently Asked Questions (FAQ)

## General

### What is beLive?
beLive is a web live audio platform designed to transform your browser into a live stage for musicians and performers. It offers features like split audio routing, in-browser vocal teleprompter, and live visual lyric presets, all without requiring an external audio interface.

### What are the main features of beLive?
Key features include browser-native audio routing (Web Audio API), Vocal Teleprompter (Bluetooth monitor), Style Mix (save/import/export presets), and Smart Mode (automated style switching, with ML planned).
